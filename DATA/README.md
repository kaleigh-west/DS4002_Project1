# How to Obtain the Data Set 
To obtain this data set, go to this link: https://www.kaggle.com/datasets/clmentbisaillon/fake-and-real-news-dataset?resource=download. From this link, click on dowload to retrieve the data files, this will dowload a zip file. Then click on the folder to un-zip the file, this will allow for access to the two CSV files, Fake.csv and True.csv.
